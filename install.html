Installing Glue

Using the Anaconda Python Distribution

We recommend using the Anaconda Python Distribution from Continuum Analytics, which makes it easy to install Glue on MacOS X, Linux, and Windows. Once Anaconda is installed, you can install Glue from the command-line/terminal by typing:

conda install -c glueviz glueviz

Other installation options

For more installation options, please take a look at the Installing and running glue section of the documentation.

Starting up glue

To start up glue from the command-line/terminal, type:

glue

on the command-line. If you run into any issues, you can open a ticket in our GitHub repository or email us at glue.viz@gmail.com.
