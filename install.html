Installing Glue

Using the Anaconda Python Distribution

We recommend using the Anaconda Python distribution from Continuum Analytics, which makes it easy to install Glue on MacOS X, Linux, and Windows. Once Anaconda is installed, you can easily install Glue from the Anaconda Launcher by cliking on Install next to the glueviz entry (the top one in the screenshot below):

Once installed, the Install button will then become a Launch button, and you can launch Glue from there. You will then also be able to start Glue by typing glue on the command-line.

Alternatively, if you are familiar with using the conda command from the terminal, you can install Glue using: conda install glueviz

Using an all-in-one Mac App

If you are using a MacOS X, you can also download an all-in-one Mac application that includes its own distribution of Python. This has the advantage that installation is as simple as copying the application to your /Applications folder, but on the other hand, Glue will then run its own internal version of Python, and you will not have access to any other Python modules you normally use.

You can download the latest Mac app here: Glue v0.6
