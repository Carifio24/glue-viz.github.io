Current Plugins

Developers are developing glue viewers as plugins for a variety of fields from astronomy to medicine. The items below are currently available on the GitHub repository and most include instructions and sample data on how to install and use. Some are experimental.

  • glue-aladin, Aladin Lite in glue
  • example_data_viewers, A collection of user-defined Glue data viewers including basketball shots, earthquakes, and Mario
  • glue-geospatial, Experimental plugin to support satellite imagery
  • glue-h5part, Experimental plugin to deal with h5part data
  • glue-jupyter, Jupyter interface for Glue
  • glue-medical, Experimental Glue medical plugin
  • glue-samp, Glue SAMP plugin
  • glue-specviz, Experimental plugin to wrap the STScI specviz tool
  • glue-vispy-viewers, Glue plugin for 3D viewers using VisPy
  • glue-wwt, Glue WorldWide Telescope plugin (experimental)

For further information on writing, installing, and using glue plugins, please see our Building Custom Data Viewers or the glueviz repository on GitHub.
